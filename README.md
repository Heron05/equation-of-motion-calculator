# Equation of Motion Calculator

A CLI-based physics calculator built in Python using OOP.
Calculates unknown variables (v, u, a, s, t) using Newton's 
equations of motion.

## Features
- Calculates all 5 kinematic variables
- Multiple formula support per variable
- Input validation and error handling

## How to Run
python class_equation_of_motion_calculator.py
